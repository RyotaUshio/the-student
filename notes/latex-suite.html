実は衝突しても大丈夫という説もある。たとえば sim は \sim に展開されてしまうので \simeq が打てない！と思っても、

{ trigger: "\\sim eq", replacement: "\\simeq", options: "mA" }, // custom

とすれば結局同じキーストロークで \simeq が実現できる。同様のテクニックは実はデフォルトのスニペットの中でも使われており、たとえば以下がある。

    { trigger: "e\\xi sts", replacement: "\\exists", options: "mA", priority: 1 },

// Insert space after greek letters and symbols, etc
    { trigger: "\\\\(${GREEK}|${SYMBOL}|${SHORT_SYMBOL})([A-Za-z])", replacement: "\\[[0]] [[1]]", options: "rmA" }

mk -> and -> A: $\cap A$

  • mk で inline math を挿入
  • and で \cap を挿入
  • この時点で $\cap$ となっているので、/\\(cap)/ にマッチしている。そして cap は SYMBOLS に含まれている。
  • ここでさらに A を打つと、trigger にマッチしてスニペットが発動し、$\capA$ が $\cap A$ となる。
  • つまり、このスニペットを使うために考えなければいけないことはほぼなく、スペースを打つ手間を知らない間に省いてくれていた！

ほかの例：

  • mk → alpha→a: $\alpha a$
{ trigger: "\\\\(${GREEK}|${SYMBOL}) hat", replacement: "\\hat{\\[[0]]}", options: "rmA" }

mkalphahat: $\hat{\alpha}$

Remark

うっかりタブストップはスニペットがトリガーされた直後のみ有効だと思っていたが、実は一度カッコ外に出た後もずっと有効っぽい。

{ trigger: "(", replacement: "(${VISUAL})", options: "mA" },

なにか数式を選択した状態で ( と打つと、選択範囲がカッコで囲われる。

Matrix shortcuts
...

Quick Latex にも同様の機能があったが、結構使いにくかった。こっちはしゅごい。

While inside a matrix, array, align, or cases environment,

  • Pressing Tab will insert the "&" symbol
  • Pressing Enter will insert "\\" and move to a new line
  • Pressing Shift + Enter will move to the end of the next line (can be used to exit the matrix)

& とスペース付きで挿入されるのは align では不便だったが、行列ではこっちのほうが便利そうだ。

align での問題も、以下のスニペットで解決できた。

    { trigger: "& =", replacement: "&= ", options: "MA" }, // custom; for better matrix shortcuts in align environmant

複数の\\が入力されてしまう問題は、次のスニペットで解決。

    { trigger: "\\\\\\\\ \\\\\\\\\\n.", replacement: "\\\\\na", options: "rMA" }, // custom; for better matrix shortcuts in align environmant

dm > align > a > Tab > = >b>Enter>Tab>=>c>Shift+Enter>Tab （スペース等一切なし）で以下を入力した上で$$...$$から脱出できる。

$$
\begin{align}
a &= b \\
 &= c
\end{align}
$$
